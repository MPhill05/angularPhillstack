Brian Magazu Contract Project

This project allows users to access a locked page with a given password from the admin. Users are greeted by name when they enter in the correct password, and the information of who is logged in is sent to the admins email. The page was designed to mimic a "MS DOS" style layout, and escapes all special characters so that users can't break into the database.





Rock'n Paradise

This is a game I developed to somewhat mimic "Tamigachi's" from the 90s. In the game, you need to feed or give love to the rock monster within 3 seconds, or he loses a life. After 3 lives are lost, he dies (it's sad). The game includes wild dinosaur screams when you feed the monster, and a background song that I wrote and produced. I plan on making the game a bit more indepth, and something where he can live a full life, rather than a quick game as it stands now.



Go Fish!

I made this project with a team as a final project while attending school at Augsburg College. This project solidified Object Oriented Programming for me, and helped drive home how full life cycle development with waterfall mehtodology operates. Recently, the project has been brought back to life, and I am developing a UI for the card game. Future updates will include additional card games, and converting it to work with Android, iOS, and a web version.
